## Why Metorial?

Metorial gives you enterprise-grade infrastructure to connect AI agents to thousands of integrations such as Salesforce, Slack, and GitHub. The core platform is fully open source and designed for scale, reliability, and a straightforward developer experience.

## Ecosystem

* [Metorial](https://github.com/metorial/metorial): 500+ high-quality MCP servers covering most all providers
* [Metorial Platform](https://github.com/metorial/metorial-platform): The core engine behind Metorial, fully open source and self-hostable
* [MCP Index](https://github.com/metorial/metorial): A curated index of 5,000+ MCP servers
* [Lowerdeck](https://github.com/metorial/lowerdeck): Shared libraries used across the stack and available for standalone use
* [Starbase](https://github.com/metorial/starbase): MCP debugging and testing utility

## SDKs

Our SDKs expose the Metorial API and include adapters for popular AI frameworks such as AI SDK or LangChain, letting you connect Metorial with a single call.
